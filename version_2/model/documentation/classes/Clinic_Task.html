addClinicTask()

addClinicTask(String  $title, String  $desc, integer  $nurses, String  $supervisor, String  $due_date, String  $due_time, integer  $clinic) : boolean

Executes a query to add a new task

This method executes a query to assign a new task to a single nurse. Tasks are assigned to nurses only by supervisors. This method inserts a a new task into the se_clinic_tasks table if it is successful.

Parameters

task title
String $desc

task description
integer $nurses

id of nurse the task is assigned to
String $supervisor

id of supervisor the task is assigned to
String $due_date

the due date of the task
String $due_time

the due time of the task
integer $clinic

id of clinic that the task is being assigned

Returns

boolean —

returns true/false indicating whether the query is successful of not
